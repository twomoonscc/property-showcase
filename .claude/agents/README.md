# agents

## a11y reviewer
Expert web accessibility auditor that reviews code changes against WCAG guidelines and ARIA specifications to identify accessibility barriers.

## code quality reviewer
Senior reviewer that analyzes code changes for clarity, naming, duplication, error handling, security, and performance, ensuring adherence to project standards.

## figma design extractor
UX/UI specialist that extracts visual specifications and layout details from Figma designs to provide implementation guidance and code examples.
